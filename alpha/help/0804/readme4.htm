许可证

  MultiPar由PAR客户端和用于控制它们的图形用户界面（GUI）组成，由Yutaka Sawada编写。尽管控制台应用程序是开源的（PAR客户端采用GPL协议），但GUI程序是闭源的。有些文章可以在我的网站上找到。 There are source code packages on GitHub.
